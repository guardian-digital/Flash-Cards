# Product Requirements Document (PRD)

## Old Town Scottsdale Flashcards



### 1. Purpose

Provide a lightweight, offline-friendly flashcard interface for tour guides and passengers.



### 2. Core Features

- Card flipping

- Prev/Next navigation

- Swipe gestures

- Highlight pill selector

- Voice narration toggle



### 3. Non-Functional Requirements

- Single-file application

- <200KB total

- No external libraries

- Instant load

- Full mobile support



### 4. Constraints

- No backend

- No build system

- No external APIs (except optional speech)



### 5. Future Enhancements

- PWA

- Multi-language

- Favorites/bookmarks
